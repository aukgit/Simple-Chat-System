Visual Studio Image Library: Animations

Use Restrictions
Animations are generally used in Windows dialogs to show indeterminate progress (progress that takes an indeterminate amount of time to complete). As part of a visual language, the images in this folder are to be used consistently, but not necessarily identical to, the usage parameters listed below:
